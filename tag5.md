# Tag 5 — GitHub Pages

**Datum:** 2026-06-05

## Was habe ich heute gelernt?

- GitHub Pages verwandelt ein Repository in eine oeffentliche Website

- Aktivierung unter Settings > Pages > Branch auswaehlen

- GitHub rendert die README.md automatisch als Startseite

- Die URL lautet: https://skunkelfree.github.io/mein-lerntagebuch

## Welches Problem hatte ich?

GitHub Pages funktioniert nur bei oeffentlichen Repos oder mit GitHub Pro.

Mein Repo ist privat - deshalb muss ich es erst auf oeffentlich stellen.

## Wie habe ich es geloest?

Unter Settings > Danger Zone > Change visibility auf Public gestellt.
